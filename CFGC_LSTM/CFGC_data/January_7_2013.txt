Workout of the Day
Warmup
2 sets w/ empty barbell:3 x Power Clean3 x Front Squat3 x Squat Clean3 x Hang Squat Clean
Dynamic stretches/warmup before each set
Workout Prep
Power Clean + Squat Clean + Hang Squat Clean

Perform 3-4 sets and build up to the weight you will start with in the workout
A guideline for starting weight would be 70-80% of your 1RM Clean

Training
Every minute on the minute for 20 minutes:Power CleanSquat CleanHang Squat Clean

Go as heavy as possible
If possible, add weight (5-10 lbs) every 5 minutes

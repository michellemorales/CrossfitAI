Workout of the Day
Warmup
3 sets of the entire Barbell complex in workout
Start with an empty barbell and add weight each set
Training
5 sets:
7 rounds of1 x Power Clean1 x Front Squat1 x Hang Squat Clean

Goal is to complete all 7 rounds without resting the bar on the ground with as much weight as possible
If you are successful, you MUST increase weight on the next set
If you fail and rest the bar on the ground, finish the set and stay at that weight for the next set
Rest 3-4 minutes b/w sets

Post heaviest successful set to comments.
